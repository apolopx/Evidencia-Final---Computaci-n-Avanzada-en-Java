IMCApp

-Descripción-

IMCApp es una aplicación web desarrollada en Java que permite a los usuarios calcular su Índice de Masa Corporal (IMC), guardar los resultados en una base de datos y consultar el historial de cálculos realizados.
La aplicación utiliza Java Servlets, JSP y JDBC, siguiendo el patrón de arquitectura MVC (Modelo - Vista - Controlador).
Los datos se almacenan en una base de datos Oracle y también se incluye un servicio REST para consultar el historial en formato JSON.


-Funcionalidades-

La aplicación permite:
* Registrar nuevos usuarios
* Iniciar sesión en el sistema
* Calcular el índice de masa corporal
* Guardar cada cálculo en la base de datos
* Consultar el historial de cálculos

  

-Tecnologías utilizadas-

* Java
* Java Servlets
* JSP
* JDBC
* Oracle SQL Developer
* Apache Tomcat
* Maven
* Jersey (REST)
* Gson (JSON)
